# vmesg
Python terminal messenger for [VK](http://vk.com)

## Requirements
* python3
* vk module (can be installed by `pip3 install vk`)

## Controls
* Ctrl-C - exit

##### If selecting dialog
* Up/Down arrows - select dialog
* Enter - enter dialog

##### If inside dialog
* Esc - return to dialog selection
